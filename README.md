# touchstone

**touch·stone** 
*/ˈtəCHˌstōn/* • *noun*
> **1.** a standard or criterion by which something is judged or recognized.
>
> **2.** a hard stone used to test the purity of gold or silver by the streak left on its surface.

**Touchstone** is a modern, configurable benchmarking library for quantum circuits.
It provides clean, scalable, and composable tools for coonstructing algorithm backed circuits for assessing transpiler and device performance.
